IA pour un Puissance 4 Problème des Huit Reines

Création d'une IA pour un jeu de Puissance 4

Description du projet :

Ce projet est une extension du projet de création d'un jeu de Puissance 4. Il a pour but d'améliorer le programme de Puissance4 de manière que l’utilisateur puisse jouer contre un ordinateur. Il a fallu dans un premier temps présenter la stratégie que notre IA allait suivre, en rédigeant un paragraphe qui en présente le détail. Dans un second temps, il a fallu programmer cette stratégie ; la mienne reposant sur l'algorithme MiniMax. Enfin, j'ai mesuré les performances de mon IA, en la comparant à celles de mes camarades.

Ressources mobilisées :

- Initiation au développement

- Introduction à l’architecture des ordinateurs

- Mathématiques discrètes

- Outils mathématiques fondamentaux

Outils utilisés :

- Langage C

Création d'un algorithme pour le problème des Huit Reines

Description du projet :

Ce projet a pour but de créer un algorithme permettant la résolution du problème des Huit Reines. J'ai dans un premier temps proposé un algorithme de type backtracking permettant de déterminer une solution au problème. Puis dans un second temps, j'ai programmé un algorithme de parcours en profondeur, qui a servi d'algorithme de comparaison, à l'algorithme de backtracking.

Ressources mobilisées :

- Graphes

- Outils numériques pour les statistiques descriptives

- Méthodes numériques

Outils utilisés :

- Python
