Création d'un site internet

Description du projet :

Ce projet, réalisé en groupe, a pour but de créer un site internet vitrine d'une association sportive. Pour mener à bien ce travail, nous avons établi le recueil des besoins du client, puis conçu l'arborescence du site, et nous avons défini une charte graphique permettant une bonne lisibilité. Nous avons ensuite réalisé le maquettage de nos pages, adapté à différents formats de visualisation avec l'outil Figma, puis programmé une version responsive avec les langages HTML et CSS. Ce travail s'est majoritairement appuyé sur l'utilisation d'outils collaboratifs, tels que Git pour versionner notre code.

Ressources mobilisées :

- Développement d’interfaces web

- Introduction à la gestion des organisations

- Bases de la communication

Outils utilisés :

- Gloomaps

- Figma

- HTML

- CSS

Gestion d'un projet

Description du projet :

Ce projet, réalisé en groupe, a pour but de gérer le projet de conception et de réalisation de la future application de gestion des réservations d'un hôtel. Nous avons analysé les besoins de l'entreprise, et rédigé un dossier de gestion de projet. Ce dossier contenait différents documents, comme la Spécification des Exigences du Logiciel, et une note de cadrage. Par la suite, nous avons intégré un organigramme des tâches, une matrice RACI, et une estimation des charges. Enfin, nous avons ajouté un diagramme de Gantt prévisionnel, et un diagramme de Gantt réel, pour mettre en évidence la différence par rapport au prévisionnel.

Ressources mobilisées :

- Introduction à la gestion des systèmes d’information

- Introduction au droit

- Anglais

- Communication technique

Outils utilisés :

- Project Libre

- Excel

- Canva
