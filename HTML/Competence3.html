Création d'un générateur de PDF

Description du projet :

Ce projet a pour but de réaliser un outil capable de produire une série de PDF, et s'est majoritairement appuyé sur Docker. En premier lieu, j'ai créé une page HTML/CSS qui servirait de modèle pour le PDF. J'ai ensuite avec PHP, extrait des informations présentes dans des fichiers .txt, pour les insérer dans ma page web. Enfin, j'ai créé un script permettant de convertir ce fichier HTML en fichier PDF.

Ressources mobilisées :

- Introduction à l’architecture des ordinateurs

- Introduction aux systèmes d’exploitation et à leur fonctionnement

- Anglais

- Bases de la communication

Outils utilisés :

- Docker

- PHP

- Bash

- HTML/CSS

Installation et configuration de services réseaux

Description du projet :

Ce projet, réalisé en groupe, a pour but d'installer et de configurer des services réseau. Nous nous sommes dans un premier temps familiarisé avec les modules PHP et MySQL présent dans notre machine virtuelle. Puis, nous avons créé des pages web, ainsi que des pages d'erreurs. Ces pages étaient présentes sur notre serveur, et nous l'avons configuré pour qu'un mot de passe soit requis pour y accéder.

Ressources mobilisées :

- Communication et fonctionnement bas niveau

- Introduction aux services réseaux

- Anglais

- Communication technique

Outils utilisés :

- Apache2

- PHP

- MySQL

- HTML

- CSS
