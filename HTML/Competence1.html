Création d'un jeu de Puissance 4

Description du projet :

Le but de ce projet était de créer un jeu de Puissance 4. Il s'est alors déroulé en deux parties : la phase de conception et la phase de réalisation. Dans un premier temps, lors de la phase de conception, j'ai conçu la maquette du jeu, j'ai élaboré un algorithme de PseudoCode, et j'ai fait la liste des procédures et des fonctions à utiliser pour mon programme. Dans un second temps, lors de la phase de réalisation, j'ai programmé en langage C mon jeu de Puissance 4, j'ai créé un cahier de tests avec les traces d'exécution, et j'ai enfin réalisé une documentation Doxygen de mon code source.

Ressources mobilisées :

- Initiation au développement

- Développement d’interfaces web

- Anglais

Outils utilisés :

- Langage C

- Doxygen

Réalisation d'interfaces IHM

Description du projet :

Ce projet, réalisé en groupe, est une extension du projet de Gestion de Projet. Il consistait à concevoir et à réaliser des interfaces IHM, pour qu'elles puissent faire partie d'une future application de gestion des réservations d'un hôtel. Il a fallu dans un premier temps réaliser un diagramme de classe et un diagramme d'activité. Puis, nous avons réalisé les maquettes de nos interfaces, ainsi qu'une charte graphique. Nous avons par la suite programmé en Java et JavaFX nos interfaces, qui devait permettre de : rechercher une réservation, rechercher et afficher les prestations servies au client, afficher et enregistrer une facture au départ du client.

Ressources mobilisées :

- Développement orienté objets

- Développement d’applications avec IHM

- Qualité de développement

- Communication technique

Outils utilisés :

- Java

- JavaFX

- UML

Création du jeu des Lettres

Description du projet :

Le but de ce projet, réalisé en classe de Première en spécialité NSI, était de créer la partie Lettres, du jeu télévisé "Des chiffres et des lettres". Dans cette partie du jeu, 2 joueurs doivent composer un mot plus long que celui de l'adversaire avec les lettres qui leur sont à disposition, pour remporter la partie. C'est un travail qui s'est appuyé sur le langage Python, ainsi que sur PyGame qui est une bibliothèque qui facilite le développement de jeux vidéo en Python. Il a fallu dans un premier temps réaliser les maquettes des images du jeu, puis programmer le jeu lui-même.

Ressources mobilisées :

- Numérique et Sciences Informatiques

- Sciences Numériques et Technologie

Outils utilisés :

- Python

- PyGame
