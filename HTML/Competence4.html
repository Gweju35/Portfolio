Création d’une base de données

Description du projet :

Ce projet, réalisé en binôme, a pour but de créer une base de données, en se basant sur "Les Rencontres Trans Musicales de Rennes". Dans un premier temps, nous avons réalisé un diagramme de classe répondant aux différentes gestions de l'événement. Nous avons ensuite traduit ce diagramme de classe vers le modèle relationnel, pour enfin l'implanter en SQL.

Ressources mobilisées :

- Introduction aux bases de données et SQL

- Mathématiques discrètes

- Introduction à l’économie durable et numérique

Outils utilisés :

- SQL

- Visual Paradigm

- MySQL Workbench

Exploitation d’une base de données

Description du projet :

Ce projet, réalisé en binôme, a pour but d'exploiter une base de données. Nous nous sommes basés sur les admissions avec ParcourSup dans toutes les formations de France en 2022. Nous avons dans un premier temps traduit et implanter un schéma relationnel. Puis, nous avons peuplé et exploité notre base de données. Pour enfin réaliser une analyse statistique des données de notre base, en effectuant une régression linéaire multiple.

Ressources mobilisées :

- Exploitation d’une base de données

- Outils numériques pour les statistiques descriptives

- Introduction à la gestion des systèmes d’information

- Anglais

Outils utilisés :

- SQL

- Python

- MySQL Workbench
